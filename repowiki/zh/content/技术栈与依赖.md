## 技术栈
- TypeScript 编写扩展逻辑，配合 VSCode Extension API 提供 TreeView、命令与配置能力
- 使用 esbuild 对 `src/extension.ts` 进行打包，输出 CommonJS 产物供扩展加载

## 构建与脚本
- `npm run watch`：监听构建，适合开发调试
- `npm run build`：生产构建，生成 `dist/extension.js`
- `npm run typecheck`：类型检查
- `npm run lint`：代码检查
